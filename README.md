# 💡 Conselhos e Avisos no Código:
- _Quando o sistema AutoGui for iniciar, não deixe o 'CapsLock' ativado; certifique-se de que ele esteja desligado._
- _Quando o sistema AutoGui for iniciar, não deixe a janela do Terminal encaixada em um dos cantos da tela; mantenha o Terminal maximizado ou em janela solta; e não utilize o modo 'F11' (tela cheia)._


- _Siga as instruções do código de forma coerente:_
> Modificar: Proibido - _É altamente recomendado que você não altere partes marcadas assim; elas são críticas para o funcionamento do sistema._
> Modificar: Negado - _Não há necessidade de modificação; há mais chances de causar erros do que melhorias, portanto é recomendado não alterar._
> Modificar: Restrito - _Siga a instrução indicada ao lado, entre parenteses._
> Modificar: Permitido - _Modificações estão liberadas, mas tenha atenção para não causar erros._
